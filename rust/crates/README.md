# Rust Crates

Planned crates:

- `HyP_core`: document model, commands, and shared types.
- `HyP_brush`: stylus samples, stroke processing, brush dynamics, and rasterization planning.
- `HyP_tiles`: tile grid, dirty tracking, cache, and stroke buffers.
- `HyP_compositor`: layer compositing and blend modes.
- `HyP_format`: native project file format.
- `HyP_export`: PNG/WebP/PSD-oriented export pipeline.
- `HyP_ffi`: Android-facing stable API boundary.
